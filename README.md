# 家政管理系统简介 python61

## 项目概述

家政管理系统（编号 S202505041）基于 Python 和 Django 框架开发，旨在提供高效、易用的家政服务管理解决方案。

## 技术栈

- Python
- Django
- MySQL
- HTML
- CSS
- JavaScript

## 运行环境

- Python 3.7
- Django 3.2
- MySQL 8.0

## 功能模块

- **首页**：系统入口，展示基础信息。
- **家政服务**：展示和管理家政服务相关信息。
- **公告信息**：发布和展示系统公告。
- **意见反馈**：收集用户反馈意见。
- **家政预约**：提供家政服务预约功能。
- **家政人员**：管理家政服务人员信息。
- **服务类型**：分类管理服务类型。
- **地区**：管理服务覆盖地区。
- **系统管理**：系统设置和管理。
- **个人中心**：用户个性化管理。

## 系统角色

系统包含以下角色：
- 系统管理员
- 家政服务人员
- 普通用户
